# openpose_restapi

### 파일 설명
openpose.py : openpose 알고리즘 파일
find_keypoint.py : openpose를 통한 좌표 도출 및 변환 실행 entry point
server.py : rest api 파일

### 실행법
python client.py 로 실행

### 결과 설명
[INDEX 번호, 한글설명, x pixel, y pixel, obj x 좌표, obj y 좌표]
